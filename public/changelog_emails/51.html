👋 Here's what we shipped this week at Rollup

 

TL;DR

 

Digital threads for visual property traceability, swap between organizations with two clicks, and import system elements via CSV files to jumpstart new workspaces.

 
 

Digital Threads

 
 

We’re revamping our digital threads. Several customers have asked us how they can see the full lineage of properties in a visual way, and interactively edit them instead of having static visualizations. Today, you can select properties from your workspace, and interactively expand and edit them. Up Next: We will work hard over the next several weeks to improve this module and add additional modules for more visual flows.

 
 

Org Switching Dropdown

 
 

Switching between organizations in Rollup is even easier now. Simply click your organization name at the top left, and select an organization you want to switch to.

 
 

Imports

 
 

You can now jumpstart your workspaces by importing system elements from a CSV file.

 

Up Next: We’re expanding this feature very soon to support full-fledged CAD assemblies, mapping everything from labels to files, and even properties such as mass, material, and bounding box dimensions.

 
 

Threaded Discussions

 
 

Convey context better with threaded discussions everywhere.

 
 

Improvements, Fixes, and Additions

 

• DOCS — Fixed the documentation portal. You can now navigate to docs.rollup.ai and after logging in, you’ll be redirected to our documentation portal.

 

• URL ATTACHMENTS — Improved the way URL attachments and favicons are cached and shown

 

• PROPERTY DEFINITIONS — Improved the updating experience by adding the ability to update their definition from any instance menu

 

• INBOX — Added inbox filters

 

• PROPERTY INSTANCES — Fixed warnings for string inputs on properties

 

• BOMS — Added BOM-specific status columns

 

• APPLICATION LEVEL — Improved workspace loading and authentication times significantly

 
 

Looking Forward

 

Rollout of our revamped integrations marketplace for productivity and engineering tools will begin next week.
